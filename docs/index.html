Syntax Overview

XanaScript uses Portuguese keywords with a C-like syntax. Every statement is a Portuguese expression.

XS
// Variables
CRIA nome = "XanaScript"
CRIA idade = 5

// Function
CHAMA ESSE CARA saudacao(nome) {
  VOLTA "Ola, " + nome
}

// Conditional
SE LIGA SO (idade >= 18) {
  SOLTA O GRITO("Adulto")
} SENAO {
  SOLTA O GRITO("Menor")
}

// Loop
REPETE NA MORAL (CRIA i = 0; i < 5; i += 1) {
  SOLTA O GRITO(i)
}

Standard Library

Modules available via IMPORTA.

HTTP Server

XS
IMPORTA { Servidor } de "http"

CRIA app = Servidor()
app.rota("/", (req) => {
  VOLTA { corpo: "Ola, mundo!" }
})
app.escutar(3000)

File System

XS
IMPORTA { lerArquivo, escreverArquivo } de "fs"

CRIA conteudo = lerArquivo("dados.txt")
escreverArquivo("saida.txt", "Novo conteudo")

ORM (TABELA)

Built-in ORM with automatic CRUD. Declare a table and get full database operations.

XS
TABELA Usuario {
  nome: TEXTO
  idade: NUMERO
}

CRIA repo = criarRepositorio(Usuario)
repo.inserir({ nome: "Joao", idade: 30 })

CRIA usuarios = repo.buscar({ idade: { $gt: 18 } })
SOLTA O GRITO(usuarios)

Macros

Compile-time code generation with MACRO.

XS
MACRO quadrado(x) {
  x * x
}

// Expands to 5 * 5 at compile time
CRIA resultado = quadrado(5)
SOLTA O GRITO(resultado) // 25

Testing

Native test runner — no external dependencies.

XS
ASSUNTO Matematica

TESTE "soma de dois numeros" {
  AFIRMA(2 + 3 == 5)
}

TESTE "subtracao" {
  AFIRMA(10 - 4 == 6)
}

// Run: xs test

Examples

HTTP Server

XS
IMPORTA { Servidor } de "http"

CRIA app = Servidor()

app.rota("/", (req) => {
  VOLTA { corpo: "Bem-vindo ao XanaScript!" }
})

app.rota("/api/dados", (req) => {
  VOLTA {
    corpo: PARSEIA('{"mensagem": "Ola", "status": 200}'),
    tipo: "application/json"
  }
})

app.escutar(3000)
SOLTA O GRITO("Servidor rodando em http://localhost:3000")

CLI Todo App

XS
IMPORTA { lerArquivo, escreverArquivo } de "fs"

CRIA ARQUIVO = "tarefas.txt"

CHAMA ESSE CARA adicionar(tarefa) {
  CRIA conteudo = lerArquivo(ARQUIVO) OU ""
  escreverArquivo(ARQUIVO, conteudo + tarefa + "\n")
  SOLTA O GRITO("Tarefa adicionada!")
}

CHAMA ESSE CARA listar() {
  CRIA conteudo = lerArquivo(ARQUIVO) OU ""
  SE LIGA SO (conteudo == "") {
    SOLTA O GRITO("Nenhuma tarefa.")
  } SENAO {
    SOLTA O GRITO("Tarefas:\n" + conteudo)
  }
}

CRIA comando = PARSEIA(AGORA VAI(process).args)[0]
CRIA arg = PARSEIA(AGORA VAI(process).args)[1]

SE LIGA SO (comando == "add") {
  adicionar(arg OU "Nova tarefa")
} SENAO SE LIGA SO (comando == "list") {
  listar()
} SENAO {
  SOLTA O GRITO("Uso: xs run todo.xs add  | list")
}

CRUD with ORM

XS
TABELA Produto {
  nome: TEXTO
  preco: NUMERO
  estoque: NUMERO
}

CRIA repo = criarRepositorio(Produto)

// Create
repo.inserir({ nome: "Teclado", preco: 199.90, estoque: 10 })

// Read
CRIA produtos = repo.buscar({})
SOLTA O GRITO(produtos)

// Update
repo.atualizar({ nome: "Teclado" }, { preco: 179.90 })

// Delete
repo.remover({ nome: "Teclado" })
